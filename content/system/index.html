AReaL
Scalable and Efficient RL Training
AReaL (Ant Reasoning RL)  is a fully open source, scalable and efficient large-scale language model learning and training system developed by Ant Research's RL Lab based on the open source project RealHF. We have made public the training details, data and infrastructure required for all models. It aims to help everyone build their own AI agents easily and economically
Easy-to-use Agentic Runtime and Environment
AWorld
AWorld (Agentic World)  aims to advance the development of agent AI by providing an easy-to-use and efficient sandbox environment with high qualilty runtimes to reduce the development and evaluation burden of agent developers. AWorld aims to achieve two main goals: simplify the work of all agent developers so that they can focus on their core value and inspire the agent creators to discover new use cases and domains by removing the unnecessary complexities
Evolving Benchmark Suite for Enhancing LLMs
ABench
ABench  is an evolving open-source benchmark suite designed to rigorously evaluate and enhance Large Language Models (LLMs) on complex cross-domain tasks. By targeting current model weaknesses, ABench provides systematic challenges in challenging and specialized domains, including but not going to be limited to physics, actuarial science, logical reasoning, law, and psychology
